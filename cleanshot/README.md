# CleanShot X

CleanShot X preferences are stored in `~/Library/Preferences/pl.maketheweb.cleanshotx.plist` and include license information, so the raw file is not tracked here.

## Restore on a new machine

1. Install CleanShot X from [cleanshot.com](https://cleanshot.com) or Setapp.
2. Enter your license key.
3. Apply the settings below manually (Settings → each tab).

## Preferred Settings

### General
- Launch at login: on
- Save screenshots to: `~/Screenshots`

### Capture
- Default action after capture: (your preference)
- Show crosshair while capturing: on

### Scrolling Capture
- Auto-scroll: on

### Cloud
- (configure as needed — avoid storing API tokens in this file)

> To back up settings: `defaults export pl.maketheweb.cleanshotx ~/dotfiles-backup-cleanshot.plist` (keep this file private).
